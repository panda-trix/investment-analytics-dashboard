# Investment Analytics Dashboard

A centralized deal-tracking and market intelligence platform built for a private credit structuring firm. The dashboard consolidates transaction data, investor details, and live market trends into a single interface, replacing fragmented spreadsheet workflows.

## Problem

The firm was managing investment deals across disconnected tools — Excel files, emails, and manually updated trackers. There was no single source of truth for deal status, investor information, or how current market conditions related to active positions.

## Solution

A full-stack web dashboard that brings together:

- **Transaction tracking** — live status of all active and historical investment deals
- **Investor management** — centralized investor profiles, commitments, and contact details
- **Market intelligence** — real-time market trend data surfaced alongside deal metrics for context-aware decision making

## Tech Stack

- **Frontend:** Replit (rapid prototyping and deployment)
- **Backend/Database:** Supabase (PostgreSQL)
- **Data processing:** Excel (for initial data import)
- **AI-assisted development:** Claude Code

## Key Features

- Single-view dashboard consolidating all deal and investor data
- Market trend integration to benchmark deals against current conditions
- Structured data model enabling fast querying across transactions and investor records
